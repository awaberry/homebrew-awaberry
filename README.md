# Awaberry Awaberry

## How do I install these formulae?

`brew install awaberry/awaberry/<formula>`

Or `brew tap awaberry/awaberry` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "awaberry/awaberry"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
